What Is AxiomCode?

AxiomCode converts natural language descriptions of algorithms into mathematically proven-correct code in Python and C. Every generated artifact comes with a cryptographic certificate of verification that can be independently audited.

Natural Language → Lean 4 Spec → Formal Proof → C Binary → Python Package
     (you describe)   (LLM)      (Lean 4)     (compiled)   (cffi binding)

Features

Natural Language Input

Describe algorithms in plain English. No programming knowledge required. The LLM generates formal Lean 4 specifications automatically.

Zero Dependencies

Pure Python stdlib + cffi only. No requests, no Flask, no SDKs. Zero attack surface from third-party libraries.

Zero-Trust Security

Every output independently verifiable. Cryptographic certificates, tamper-evident audit logs, encrypted key store, sandboxed execution.

Multi-Backend LLMs

Ollama (local), Mistral, OpenAI, Anthropic. No SDKs — direct HTTP calls. LLM caching reduces cost and latency.

Proof Visualization

2D port graphs, force-directed dependency graphs, 3D spatial layouts. Interactive, self-contained HTML with D3.js.

Version Management

Automatic upgrades/downgrades with data migration. Backup before every change. One-command rollback. History retention.

Commercial Licensing

HMAC-signed licenses bound to hardware. Three tiers: Community (free), Pro ($49/mo), Enterprise (custom). Offline verification.

Persistence Layer

JSON-based storage with schema versioning. Atomic writes prevent corruption. Full history retention across version changes.

Quick Start

1. Install

git clone https://github.com/nrupala/axiomcode.git
cd axiomcode
pip install -e ".[dev]"

2. Generate Your First Verified Algorithm

# Quick generate
python cli.py "implement binary search on a sorted array"

# Interactive guided mode
python cli.py guide

# Browse examples
python cli.py examples

3. View the Proof

# 2D port graph view
python cli.py visualize binary_search --mode 2d

# Force-directed graph
python cli.py visualize binary_search --mode force

4. Manage Licenses

# Generate root key pair (do this ONCE)
python cli.py license keygen

# Issue a license
python cli.py license issue --user user@example.com --name "User" --tier pro

# Verify a license
python cli.py license verify --license-file user.license.json

Security Model

Zero-Trust

Every output is independently verifiable. No blind trust in any component.

Encrypted at Rest

Keys encrypted with PBKDF2-derived master key (600,000 iterations).

Tamper-Evident

Hash-chained audit log detects any modification to historical records.

Signed Artifacts

HMAC signatures on all binaries, certificates, and licenses.

Sandboxed Execution

Isolated subprocess with restricted environment for untrusted code.

No Telemetry

Zero data collection, zero tracking. Your data stays on your machine.
